Gopi is a .NET library for working with emails through SMTP using
a local storage provider system for keeping track of failed emails
to resend them. Emails are serialized and deserialized to be
stored in storage system using XML format.

The implementaiton is described in details on my blog:

http://keyvan.io/gopi-smtp-component-for-net-2-0
http://keyvan.io/how-to-serialize-a-mailmessage
